AdminLTE Tailwind

A modern, fully responsive admin dashboard template built with Tailwind CSS v4. Clean design, modular components, and easy customization.

View Dashboard GitHub

Features

Tailwind CSS v4

Built with the latest Tailwind CSS v4, utilizing the new CSS-based configuration system.

Fully Responsive

Mobile-first design that looks great on all devices from phones to large desktops.

Modular Components

Reusable UI components including cards, buttons, alerts, badges, forms, and more.

Vite Powered

Lightning-fast development with Vite's instant hot module replacement (HMR).

Easy Customization

Customize colors, typography, and spacing easily with Tailwind's theme system.

Production Ready

Optimized build output with minified CSS and JavaScript for production deployment.

Quick Start

1. Clone the repository

git clone https://github.com/ColorlibHQ/adminlte-tailwind.git
cd adminlte-tailwind

2. Install dependencies

npm install

3. Start the development server

npm run dev

4. Build for production

npm run build
